Требования к системе:
Пользователь может войти в систему 2 способами:
Без реферальной ссылки
По реферальной ссылке (При входе по реферальной ссылке его нужно занести в direct partners к пользователю чью ссылку он использовал)
Пользователь может инвестировать
При инвестировании 5% от суммы должно зачисляться на счет контракта
У пользователей присутствуют уровни зависящие от суммы инвестиций
1й уровень - 0.005 eth
2й уровень - 0.01 eth
3й уровень - 0.02 eth
4й уровень - 0.05 eth
5й уровень - 0.1 eth
6й уровень - 0.2 eth
7й уровень - 0.5 eh
8й уровень -  1 eth
9й уровень - 2 eth
10й уровень - 5 eth
Пользователь может посмотреть свой уровень
Пользователь может посмотреть кол-во direct partners и их уровень
Пользователь может вывести свои деньги
Пользователь автоматически получает комиссию когда кто-то из его partners выводит деньги, если его уровень >= глубине partner

Доп требования:
Депозит только в своем токене (нужно написать контракт ERC20, задеплоить его и принимать депозит только в данных токенах)
Добавить любую функцию с поддержкой подписи EIP712
https://eips.ethereum.org/EIPS/eip-712
